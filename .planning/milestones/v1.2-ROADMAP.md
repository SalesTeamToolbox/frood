## Milestone Summary

**Key Decisions:**

- GitHub token via env var reference (not embedded in .mcp.json) — secrets stay out of repo
- Single .mcp.json config for all MCP servers — simple, standard pattern
- PreToolUse gate for security files — blocks before edit, PostToolUse alerts after
- Skills use inline templates — self-contained, no external template dependency
- Agent definitions follow plain markdown format — consistent with existing agents
- jcodemunch guidance per work-type — context-loader detects type and emits relevant guidance

**Issues Resolved:**

- Stale .playwright-mcp artifact directory cleaned up
- security-monitor.py refactored to use shared security_config.py registry
