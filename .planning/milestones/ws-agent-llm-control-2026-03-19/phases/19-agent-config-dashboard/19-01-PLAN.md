**3. Shared routing helper functions (insert near the data loaders):**

`routingSelect(field, label, currentValue, isInherited, inheritedFrom)`:
- Renders a `<select>` with `<optgroup>` headers for each tier ("L1 Models", "Fallback Models", "L2 Premium Models")
- First option: `<option value="">Use default (Inherit)</option>`
- Each model option shows health dot + display name + provider
- Health dot: green circle HTML entity for "healthy", yellow for "degraded", red for "unhealthy" (use Unicode circles or HTML entities, not emoji -- emoji may not render in all selects)
- Only render `<optgroup>` when the tier array has items (avoid empty optgroup -- Research Pitfall 3)
- Inherited values: muted/gray text color on the select element, small "inherited from {source}" label below
- Overridden values: normal text color, reset (X) button that calls `clearRoutingField(field)`
- onchange: calls `updateRoutingEdit(field, this.value)`
- Use `esc()` on ALL interpolated values for XSS protection
- Use `m.key` for option value, `m.display_name` for display text (Research Pitfall 1)

`renderChainSummary(chain)`:
- Takes resolution_chain array from API (each entry has: field, value, source)
- Renders inline badge spans connected with arrow entities
- Source styling: profile override (source starts with "profile:") = teal background, _default source = gold background, FALLBACK_ROUTING = muted gray
- Source labels: "overridden", "default", "system"
- Return empty string if chain is empty/null/undefined

`updateRoutingEdit(field, value)`:
- If value is empty string, still store it (means "clear override / inherit")
- Sets `state.routingEdits[field] = value`
- Calls `renderSettingsPanel()` to re-render with dirty state

`clearRoutingField(field)`:
- Sets `state.routingEdits[field] = ""` (empty = clear override)
- Calls `renderSettingsPanel()`

`saveRouting(profileName)`:
- Sets `state.routingSaving = true`, re-renders
- Builds body from the appropriate edits object (for _default: state.routingEdits)
- For each field in edits: if value is empty string, send as null in body (to clear the override). If undefined, omit (no change).
- If no fields to send, toast "No changes to save" and return
- Calls `PUT /api/agent-routing/${encodeURIComponent(profileName)}` via `api()` helper
- On success: clears edits object, reloads routing config via `loadRoutingConfig()`, toasts "Routing updated. Takes effect on next dispatch."
- On error: toasts error message
- Clears saving flag, re-renders appropriate section

`resetRouting(profileName)`:
- Confirms via `confirm()` with descriptive message ("Reset global routing defaults? This will clear all overrides.")
- Calls `DELETE /api/agent-routing/${encodeURIComponent(profileName)}` via `api()` helper
- On success: clears edits, reloads routing config, toasts success
- On 404: toasts "No overrides to reset" (not an error)
- On other error: toasts error
