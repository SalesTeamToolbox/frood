<objective>
Create the Stop hook entry point and background worker that extract structured learnings from Claude Code sessions and store them in Qdrant's `agent42_knowledge` collection.

Purpose: Enable Agent42 to automatically learn from every CC session — decisions, corrections, patterns — without any user action. The hook fires at session end, the worker calls Agent42's API for LLM extraction, then embeds and upserts to Qdrant.

Output: Three files — hook entry point, background worker, comprehensive test suite.

   extract_data = {

       "files_modified": files_modified,
       "session_summary": session_summary,
       "project_dir": project_dir,

`make_point_id(content: str) -> str`:
```python
return str(uuid.uuid5(NAMESPACE, f"knowledge:{content}"))
```

`call_extraction_api(extract_data: dict) -> list[dict]`:
- POST to `{AGENT42_DASHBOARD_URL}/api/knowledge/learn` with extract_data
- Use `urllib.request.Request` + `urlopen(req, timeout=30)`
- On success: parse response JSON, return `result.get("learnings", [])`
- On any exception (ConnectionRefusedError, URLError, timeout): return empty list (silent failure)

`dedup_or_store(store: QdrantStore, embedder, learning: dict, session_id: str) -> str`:
- Embed `learning["content"]` using `embedder.encode(learning["content"][:2000])`
- Search KNOWLEDGE collection: `store.search_with_lifecycle(QdrantStore.KNOWLEDGE, vector, top_k=3, exclude_forgotten=True)`
- For each hit: if `hit["raw_score"] >= SIMILARITY_THRESHOLD` → call `store.strengthen_point(QdrantStore.KNOWLEDGE, hit["point_id"], boost=0.1)` → return `"boosted"`
- If no match above threshold: create new point:
  ```python
  point_id = make_point_id(learning["content"])
  payload = {
      "text": learning["content"],
      "source": "knowledge_learn",
      "learning_type": learning["learning_type"],
      "category": learning["category"],
      "title": learning["title"],
      "confidence": learning["confidence"],
      "recall_count": 0,
      "status": "active",
      "session_id": session_id,
      "timestamp": time.time(),
  }
  point = PointStruct(id=point_id, vector=vector, payload=payload)
  store._ensure_collection(QdrantStore.KNOWLEDGE)
  collection_name = store._collection_name(QdrantStore.KNOWLEDGE)
  store._client.upsert(collection_name=collection_name, points=[point])
  ```
  Return `"stored"`

`process_learnings(extract_file: str) -> None`:
Main worker function:
1. Read temp file from `sys.argv[1]`, parse JSON as `extract_data`
2. Delete temp file after reading (cleanup)
3. Call `call_extraction_api(extract_data)` → get list of learnings
4. If empty list → update status, return
5. Initialize ONNX embedder: `_find_onnx_model_dir()` → `_OnnxEmbedder(model_dir)`
6. Initialize Qdrant: `QdrantConfig(url=qdrant_url, local_path=qdrant_local, vector_dim=384)` → `QdrantStore(config)`
7. Check `store.is_available` → if not, return silently
8. Generate `session_id = str(uuid.uuid4())`
9. For each learning in learnings list: call `dedup_or_store(store, embedder, learning, session_id)`
10. Update status file with counts

Entry point:
```python
if __name__ == "__main__":
    try:
        if len(sys.argv) < 2:
            sys.exit(0)
        process_learnings(sys.argv[1])
    except Exception:
        pass  # Top-level silence — never crash
```
