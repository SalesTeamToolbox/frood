<objective>
Register Mistral as Agent42's first dual-provider integration: Codestral (free code endpoint) and La Plateforme (credits-based general endpoint), with accurate pricing for both.

Purpose: Mistral exposes two separate API endpoints with two separate API keys. Codestral (`codestral.mistral.ai`) is a genuinely free code-specialist endpoint (30 RPM). La Plateforme (`api.mistral.ai`) provides mistral-large and mistral-small at per-token pricing (CHEAP tier). Both are OpenAI-compatible. This phase registers both providers, three models, and correct SpendingTracker pricing to prevent false spend-cap trips on Codestral or wild overestimation on La Plateforme.

Output: Modified `providers/registry.py` with 2 ProviderSpecs, 3 ModelSpecs, and 3 pricing entries. Modified `core/config.py` with 2 new API key fields. Updated `.env.example` with documentation for both keys.
</objective>

class ProviderSpec:

    base_url: str
    api_key_env: str
    display_name: str
    default_model: str = ""
