**Printful status mapping constant** (per D-10):
```python
PRINTFUL_STATUS_MAP = {
    "pending": "submitted",
    "in_production": "printing",
    "fulfilled": "shipped",
    "package_shipped": "shipped",
    "delivered": "delivered",
    "canceled": "canceled",
}
```

**Main endpoint** — `POST /webhook`:
```python
@router.post("/webhook")
async def printful_webhook(request: Request) -> dict:
```

Implementation steps:
1. Read raw body: `payload = await request.body()`
2. Parse JSON: `data = json.loads(payload)` — raise HTTPException 400 on JSONDecodeError
3. Extract event info:
   - `event_type = data.get("type", "")`
   - `order_data = data.get("data", {}).get("order", {})`
   - `external_id = order_data.get("external_id", "")`
4. If no `external_id`, return `{"status": "ignored", "reason": "no external_id"}`
5. Build idempotency key (per Research Pitfall 5 — composite key to allow multiple status transitions):
   - For `package_shipped`: `shipment = data.get("data", {}).get("shipment", {})`, key = `f"package_shipped:{external_id}:{shipment.get('tracking_number', '')}"`
   - For all others: `order_status = order_data.get("status", "")`, key = `f"{event_type}:{external_id}:{order_status}"`
6. Open DB connection: `async with aiosqlite.connect(str(database.DB_PATH)) as db:`
7. Check idempotency: `SELECT 1 FROM printful_events_processed WHERE event_key = ?` — if exists, return `{"status": "already_processed"}`
8. Route by event_type:
   - `"order_updated"`: call `await _handle_order_updated(db, external_id, order_data)`
   - `"package_shipped"`: call `await _handle_package_shipped(db, external_id, data.get("data", {}))`
9. Record processed event: `INSERT INTO printful_events_processed (event_key, event_type) VALUES (?, ?)`
10. Commit and return `{"status": "processed"}`

**Handler functions:**

`_handle_order_updated(db, external_id, order_data)`:
- Get Printful status: `pf_status = order_data.get("status", "")`
- Map to app status: `app_status = PRINTFUL_STATUS_MAP.get(pf_status, "")` — if empty string, log warning and return
- Call `await order_service.update_order_status(external_id, app_status)`
- Log: `logger.info("Order %s status updated to %s (Printful: %s)", external_id, app_status, pf_status)`

`_handle_package_shipped(db, external_id, event_data)`:
- Extract shipment: `shipment = event_data.get("shipment", {})`
- Extract tracking: `tracking_number = shipment.get("tracking_number", "")`, `tracking_url = shipment.get("tracking_url", "")`, `carrier = shipment.get("service", "")`
- Update order in DB — execute direct SQL UPDATE (because `update_order_status` doesn't handle tracking fields):
```python
await db.execute(
    """UPDATE orders
       SET status = 'shipped',
           tracking_url = ?,
           tracking_number = ?,
           updated_at = CURRENT_TIMESTAMP
       WHERE id = ?""",
    (tracking_url, tracking_number, external_id),
)
await db.commit()
```
- Look up user email for shipping notification:
```python
cursor = await db.execute(
    "SELECT u.email FROM users u JOIN orders o ON o.user_id = u.id WHERE o.id = ?",
    (external_id,),
)
email_row = await cursor.fetchone()
```
- If email found, fetch the order dict and call `send_shipping_confirmation()`:
```python
db.row_factory = aiosqlite.Row
cursor = await db.execute("SELECT * FROM orders WHERE id = ?", (external_id,))
order_row = await cursor.fetchone()
if order_row and email_row:
    order = dict(order_row)
    await order_service.send_shipping_confirmation(
        user_email=email_row[0],
        order=order,
        tracking_url=tracking_url,
        tracking_number=tracking_number,
        carrier=carrier,
    )
```
- Log: `logger.info("Order %s shipped — tracking: %s", external_id, tracking_number)`
