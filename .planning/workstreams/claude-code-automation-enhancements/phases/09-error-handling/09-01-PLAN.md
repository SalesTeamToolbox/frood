### 2. Add FastAPI Exception Handler in `dashboard/server.py`

<task>
<description>Add global exception handler for unified error responses</description>
<expected_outcome>
- @app.exception_handler registered for HTTPException
- All error responses follow {error, message, action} format
- Existing endpoints continue to work
</expected_outcome>
</task>

**In `dashboard/server.py`:**

```python
from fastapi import Request
from fastapi.responses import JSONResponse
from core.error_codes import classify_error, ERROR_MESSAGES, ErrorCode

@app.exception_handler(HTTPException)
async def unified_error_handler(request: Request, exc: HTTPException):
    """Convert exceptions to user-friendly error responses."""
    error_code = classify_error(exc)
    error_info = ERROR_MESSAGES.get(error_code, ERROR_MESSAGES[ErrorCode.UNKNOWN])

    return JSONResponse(
        status_code=exc.status_code,
        content={
            "error": error_code.value,
            "message": error_info["message"],
            "action": error_info["action"]
        }
    )
```

```javascript
// Loading indicator using only safe DOM APIs
window.LoadingIndicator = class LoadingIndicator {
  constructor(element) {
    this.element = element;
    this.spinner = document.createElement('div');
    this.spinner.className = 'spinner';
    this.spinner.style.display = 'none';
    this.element.appendChild(this.spinner);
  }

  show() { this.spinner.style.display = 'inline-block'; }
  hide() { this.spinner.style.display = 'none'; }
};

window.showError = function(code, message, action) {
  // Safe toast creation without innerHTML
  const toast = document.createElement('div');
  toast.className = 'toast toast-error';

  const title = document.createElement('h4');
  title.textContent = message;
  toast.appendChild(title);

  const desc = document.createElement('p');
  desc.textContent = action;
  toast.appendChild(desc);

  document.getElementById('toasts').appendChild(toast);
  setTimeout(() => toast.remove(), 5000);
};
```
