       and symbol distribution. Focus your file reads on areas relevant to your
       focus area instead of scanning everything.

       ## Repo Outline
       {JCODEMUNCH_OUTLINE}

       ## File Tree
       {JCODEMUNCH_TREE}
       </jcodemunch_prefetch>
       ```

    **If jcodemunch is NOT available:**

       ```

       ### {file_path} (outline)
       {get_file_outline output — classes, functions, signatures}

       {get_symbol output — exact function/class source code}
       ```
       </implementation_targets>
       ```
