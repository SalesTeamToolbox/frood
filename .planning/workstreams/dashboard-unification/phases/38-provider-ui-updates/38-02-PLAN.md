Function 2 - `loadProviderStatus()`: Calls `api("/settings/provider-status")`. Sets `state.providerStatus` on success, null on error. Toggles `state.providerStatusLoading`. Follows the exact `loadOrStatus()` pattern.

Function 3 - `loadAgentModels(provider)`: Gets `#agent-model` select element. Sets its content to "Loading models..." placeholder. Calls `api("/agents/models")`. Extracts `allModels[provider]` entries. For each entry, creates an option element with value = model ID and text = `category -- modelId` (using esc() on both values). On error, shows "(failed to load models)" option.
