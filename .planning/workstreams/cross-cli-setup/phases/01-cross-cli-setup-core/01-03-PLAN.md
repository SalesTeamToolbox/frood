    CliAdapter ABC:
    ```python
    class CliAdapter(ABC):
        name: str  # "claude-code" or "opencode"
        @abstractmethod
        def detect(self) -> dict: ...  # {"installed": bool, "wired": bool, ...}
        @abstractmethod
        def wire(self) -> dict: ...  # {"changed": bool, "backup": Optional[Path], "targets": [...]}
        @abstractmethod
        def unwire(self) -> dict: ...  # {"changed": bool, "targets": [...]}
    ```

    ClaudeCodeSetup:
    - `__init__(self, root: Optional[Path] = None)` — root defaults to `Path.home()`. Target = `root / ".claude" / "settings.json"`.
    - `detect()`:
      - `installed` = True iff settings.json parent dir exists.
      - `wired` = True iff settings.json exists AND `settings["mcpServers"].get("frood")` present.
    - `wire()`:
      - Build `mcp_entry = copy.deepcopy(FROOD_MCP_ENTRY_CLAUDE)`.
      - Load existing settings (or `{}` if absent). `existing_mcp = settings.get("mcpServers", {})`.
      - Deep-compare — if `existing_mcp.get("frood") == mcp_entry`: return `{"changed": False, "backup": None}` (idempotent).
      - Otherwise: `_write_backup_if_first(target)` → merges mcp_entry in, writes atomically.
    - `unwire()`:
      - If settings.json missing or "frood" not in mcpServers: return `{"changed": False}`.
      - Remove the "frood" key only. If mcpServers is now empty, drop the mcpServers key itself ONLY if it was created by a prior wire() into an absent file. Track via backup: if the backup file exists, deep-diff the backup against current to decide. Simpler and safer: leave empty mcpServers dict alone — it's still byte-different from a pre-existing file with no mcpServers key.
      - **For true byte-identity**: when unwire detects a backup file matches the pre-wire state, prefer restoring from backup (copy `.bak-<ts>` back onto target, then delete the backup). Document this: "unwire restores from the latest .bak-* sibling if present; else performs in-place removal."

    OpenCodeSetup:
    - `__init__(self, project_paths: Optional[list[Path]] = None, manifest: Optional[dict] = None)`.
      - If project_paths not given: use manifest['clis']['opencode']['projects'] from load_manifest(). If that's "auto" or missing, do a shallow scan of a default project root (use cwd's parent — document this; per Claude's Discretion).
    - `detect()`: for each path, `installed` = opencode.json exists; `wired` = "frood" in its `mcp` dict. Return `{"installed": bool, "wired": bool, "projects": [{path, installed, wired}, ...]}`.
    - `wire()`:
      - For each project path:
        - Target A: opencode.json — merge FROOD_MCP_ENTRY_OPENCODE into `data["mcp"]["frood"]`. Same
          idempotency check + backup-on-first-write as Claude Code.
        - Target B: AGENTS.md — if marker block already present, no-op. Else append the block
          (create file if absent). Take a backup of AGENTS.md if pre-existing.
    - `unwire()`:
      - For each project path:
        - opencode.json: if backup exists and is pre-wire, restore from backup. Else remove "frood" key in place.
        - AGENTS.md: remove everything between MARKER_BEGIN and MARKER_END (inclusive of the markers and the single trailing newline). If the file is now empty AND was created by wire (detect via backup presence of the original state — if there's no .bak-*, we created the file; delete it). Byte-identical: if we have the backup, restore it.

    Backup helper (private):
    ```python
    def _write_backup_if_first(target: Path) -> Optional[Path]:
        """Write target.bak-<ts> next to target, but only if no prior backup exists."""
        if not target.exists():
            return None
        # Idempotency: if ANY .bak-* sibling exists for this target, skip (first-write-only per SAFE-01)
        siblings = list(target.parent.glob(f"{target.name}.bak-*"))
        if siblings:
            return siblings[0]  # return the original backup for traceability
        ts = datetime.datetime.now().strftime(BACKUP_SUFFIX_FMT)
        bak = target.with_name(f"{target.name}.bak-{ts}")
        shutil.copy2(target, bak)
        return bak
    ```

    ```python
        """Dispatch to the right adapter based on cli name."""

    def unwire_cli(cli: str, manifest: Optional[dict] = None) -> dict:

    ```
