def resolve_model(provider: str, task_category: str, tier: str = "") -> str:
    """Resolve the best model for a provider + task category.
    Falls back to 'general' if the task category isn't mapped for the given provider.
    """
    effective_category = _TIER_CATEGORY_UPGRADE.get(tier, task_category)
    models = PROVIDER_MODELS.get(provider, PROVIDER_MODELS.get("anthropic", {}))
    return models.get(effective_category, models.get("general", "claude-sonnet-4-6"))

    Key implementation details:

    1. **RoutingDecision dataclass** (per D-12): frozen dataclass with fields: `provider`, `model`, `tier`, `task_category`, `base_category`, `cost_estimate`. The `base_category` field captures the pre-upgrade category for log observability (success criterion 1).

    2. **_ROLE_CATEGORY_MAP** (per D-01, D-03): Static dict constant at class level. Exact mapping: `{"engineer": "coding", "researcher": "research", "writer": "content", "analyst": "strategy"}`. No prefix matching per D-03.

    3. **_MODEL_PRICING** (per D-10): Static dict constant with per-token costs (input_per_token, output_per_token) tuples. Include all models from PROVIDER_MODELS dict plus _PRICING_FALLBACK = (5.0/1_000_000, 15.0/1_000_000). Values from RESEARCH.md pricing table.

    4. **__init__(self, reward_system=None, tier_determinator=None)**: Store both. If tier_determinator is None, create a new TierDeterminator() instance.

    5. **async def resolve(self, role, agent_id, preferred_provider="") -> RoutingDecision** (per D-12):
       - If reward_system is None: tier = "", obs_count logic skipped
       - Otherwise: score = await self._reward_system.score(agent_id); tier = self._tier_determinator.determine(score, obs_count=0) (per Pitfall 1 recommendation: use 0 as safe default, add TODO comment about wiring real obs_count in Phase 27)
       - base_category = _ROLE_CATEGORY_MAP.get(role or "", "general") (per D-02)
       - provider = preferred_provider if preferred_provider else ("synthetic" if os.environ.get("SYNTHETIC_API_KEY") else "anthropic") (per D-06/D-08)
       - model = resolve_model(provider, base_category, tier) -- imported from core.agent_manager (per D-05, no new upgrade logic)
       - task_category = _TIER_CATEGORY_UPGRADE.get(tier, base_category) -- imported from core.agent_manager for logging observability
       - Return RoutingDecision with all fields, cost_estimate=0.0 (per D-09)

    6. **def estimate_cost(self, model, input_tokens, output_tokens) -> float** (per D-10):
       - price = _MODEL_PRICING.get(model, _PRICING_FALLBACK)
       - return round(input_tokens * price[0] + output_tokens * price[1], 8)
