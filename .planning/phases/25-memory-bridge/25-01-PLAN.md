**Method: `async def recall()`** (MEM-01, MEM-02, D-02, D-03, D-10):
- Parameters: `query: str, agent_id: str, company_id: str = "", top_k: int = 5, score_threshold: float = 0.25`
- If `not agent_id`: raise `ValueError("agent_id is required for recall() — never search unfiltered")` (D-10)
- If `not self.memory_store` or `not self.memory_store.embeddings.is_available`: return `[]`
- If `not self.memory_store._qdrant` or `not self.memory_store._qdrant.is_available`: return `[]`
- Get embedding: `query_vector = await self.memory_store.embeddings.embed_text(query)` (async, wraps ONNX in to_thread)
- Build FieldCondition filters for `agent_id` (always) and `company_id` (if non-empty):
  ```python
  from qdrant_client.models import FieldCondition, MatchValue, Filter
  conditions = [FieldCondition(key="agent_id", match=MatchValue(value=agent_id))]
  if company_id:
      conditions.append(FieldCondition(key="company_id", match=MatchValue(value=company_id)))
  ```
- Call `self.memory_store._qdrant.search()` directly with the query_vector and filters. Do NOT go through `semantic_search()` — it lacks agent_id filter support (per research open question 1 recommendation).
  Since `QdrantStore.search()` does not accept arbitrary FieldCondition lists, call `_client.query_points()` directly:
  ```python
  qdrant = self.memory_store._qdrant
  name = qdrant._collection_name(qdrant.MEMORY)
  qdrant._ensure_collection(qdrant.MEMORY)
  query_filter = Filter(must=conditions)
  response = qdrant._client.query_points(
      collection_name=name,
      query=query_vector,
      limit=top_k,
      query_filter=query_filter,
  )
  ```
- Also search HISTORY collection with same filters, merge results by score, take top_k.
- Filter results where `score < score_threshold`.
- Return list of dicts: `[{"text": ..., "score": ..., "source": ..., "metadata": {...}}]` — matching `MemoryItem` shape.
- Wrap all Qdrant calls in `try/except Exception` returning `[]` on failure (graceful degradation).
