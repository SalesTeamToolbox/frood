```typescript
import type {
  SidecarExecuteRequest,
  SidecarExecuteResponse,
  SidecarHealthResponse,
  MemoryRecallRequest,
  MemoryRecallResponse,
  MemoryStoreRequest,
  MemoryStoreResponse,
} from "./types.js";

export class Agent42Client {
  constructor(
    private readonly baseUrl: string,
    private readonly bearerToken: string,
  ) {}
```

Methods:

1. `private authHeaders(): Record<string, string>` — returns Content-Type: application/json + Authorization: Bearer {token} (omit Authorization if bearerToken is empty)

2. `async execute(body: SidecarExecuteRequest): Promise<SidecarExecuteResponse>` — POST to `{baseUrl}/sidecar/execute` with authHeaders, 30_000ms timeout (D-17). Check `resp.ok` (covers 200-299 including 202). Throw on non-2xx. NO retry (D-18).

3. `async health(): Promise<SidecarHealthResponse>` — GET to `{baseUrl}/sidecar/health` with NO auth header (public endpoint per D-05), 5_000ms timeout (D-17). Retry once on 5xx with 1s delay (D-18).

4. `async memoryRecall(body: MemoryRecallRequest): Promise<MemoryRecallResponse>` — POST to `{baseUrl}/memory/recall` with authHeaders, 10_000ms timeout (D-17). Retry once on 5xx with 1s delay (D-18).

5. `async memoryStore(body: MemoryStoreRequest): Promise<MemoryStoreResponse>` — POST to `{baseUrl}/memory/store` with authHeaders, 10_000ms timeout (D-17). Retry once on 5xx with 1s delay (D-18).

6. `private async fetchWithTimeout(url: string, init: RequestInit, timeoutMs: number): Promise<Response>` — AbortController + setTimeout pattern. Clear timeout in finally block.

7. `private async fetchWithRetry(url: string, init: RequestInit, timeoutMs: number): Promise<Response>` — Calls fetchWithTimeout. On 5xx, waits 1s (`await new Promise(r => setTimeout(r, 1000))`), retries once. Throws on second failure.
