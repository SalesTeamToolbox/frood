    1. `build_parser()` — argparse.ArgumentParser with:
       - `--agent42-db` (required): Path to source effectiveness.db
       - `--qdrant-url` (required): Source Qdrant server URL
       - `--target-qdrant-url` (required): Target Qdrant server URL
       - `--paperclip-company-id` (required): Target Paperclip company UUID
       - `--collection-prefix` (default "agent42"): Qdrant collection prefix
       - `--batch-size` (int, default 100): Points per scroll batch
       - `--dry-run` (store_true): Validate without writing
       - `--target-db` (optional): Target effectiveness.db path (default: same as --agent42-db with ".migrated" suffix)

    2. `NAMESPACE = uuid.UUID("a42a42a4-2a42-4a42-a42a-42a42a42a42a")` — same namespace as qdrant_store.py line 248

    3. `remap_point(point, target_company_id: str) -> PointStruct`:
       - Copy payload dict, set payload["company_id"] = target_company_id
       - Preserve payload["agent_id"] as-is (D-08)
       - Regenerate UUID5 point ID: `content = f"{payload.get('source', '')}:{payload.get('text', '')}"` then `str(uuid.uuid5(NAMESPACE, content))` (D-10)
       - Return PointStruct(id=new_id, vector=point.vector, payload=payload)

    4. `async def migrate_collection(src_client, dst_client, collection: str, company_id: str, batch_size: int, dry_run: bool) -> int`:
       - Scroll loop using `while True` pattern (Pitfall 1 from RESEARCH.md):
         ```python
         offset = None
         total = 0
         while True:
             records, next_offset = src_client.scroll(
                 collection_name=collection, offset=offset, limit=batch_size,
                 with_vectors=True, with_payload=True,
             )
             if not records:
                 break
             points = [remap_point(r, company_id) for r in records]
             if not dry_run:
                 dst_client.upsert(collection_name=collection, points=points)
             total += len(points)
             if next_offset is None:
                 break
             offset = next_offset
         return total
         ```

    5. `async def ensure_target_collections(dst_store: QdrantStore, prefix: str)`:
       - Call dst_store._ensure_collection(suffix) for each of QdrantStore.MEMORY, QdrantStore.HISTORY, QdrantStore.CONVERSATIONS, QdrantStore.KNOWLEDGE
       - Ensures payload indexes (agent_id, company_id as tenants) are created (Pitfall 6)

    6. `async def migrate_effectiveness(src_db: str, dst_db: str) -> dict`:
       - Instantiate EffectivenessStore(dst_db) and call `await store._ensure_db()` (Pitfall 4)
       - Read ALL rows from source tool_invocations, routing_decisions, spend_history, run_transcripts tables
       - INSERT OR IGNORE into target for each table
       - Return dict with row counts per table: {"tool_invocations": N, "routing_decisions": N, "spend_history": N, "run_transcripts": N}

    7. `async def run_migration(args) -> None`:
       - Validate --agent42-db path exists (Path(args.agent42_db).exists() or exit 1)
       - Create source QdrantClient(url=args.qdrant_url) and target QdrantClient(url=args.target_qdrant_url)
       - Create target QdrantStore and call ensure_target_collections()
       - For each collection suffix in [MEMORY, HISTORY, CONVERSATIONS, KNOWLEDGE]:
         - collection_name = f"{args.collection_prefix}_{suffix}"
         - Call migrate_collection(..., dry_run=args.dry_run)
         - Log: "Migrated {N} points from {collection_name}"
       - If not dry_run: call migrate_effectiveness(args.agent42_db, target_db_path)
       - Log summary: total points migrated, total effectiveness rows per table

    8. `if __name__ == "__main__":` block:
       ```python
       logging.basicConfig(level=logging.INFO, format="%(asctime)s %(levelname)s %(message)s")
       args = build_parser().parse_args()
       asyncio.run(run_migration(args))
       ```

    Per D-02: All arguments are flags with required=True (no interactive prompts). Per D-03: Module reuses existing qdrant_store and effectiveness imports.
